# Good Example — pptx-ai-slide 스킬

## 사용자 요청

"고객 문의 자동 분류 에이전트를 C-Level에 피칭할 발표 자료를 만들어줄래? 핵심은 비용 절감과 응답 속도 개선이고, 7장 이내로."

## 승인 이유

- 에이전트 프로젝트의 피치 덱 필요
- 청중(C-Level)에 맞는 메시지 밀도 조정
- 분량 명확 (7장)

## 예상 처리

1. 스토리라인: 문제(현 프로세스 비효율) → 솔루션(에이전트) → 아키텍처 → 데모 → 비용ROI → HITL → 로드맵
2. 슬라이드 구성: 각 슬라이드 1메시지 원칙
3. 비주얼: 비용 비교 차트, 응답 시간 개선 그래프
4. HITL 슬라이드: 에스컬레이션 플로우
5. PPTX 레이아웃: 템플릿 마스터 유지, 텍스트 overflow 0건

## 최종 결과물

7장 C-Level 피치 덱으로 경영진을 설득할 수 있는 프레젠테이션
