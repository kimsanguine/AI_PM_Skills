# Bad Example — agent-demo-video 스킬

## 사용자 요청

"에이전트 영상 만들어줄래?"

## 거절 이유

- 어떤 에이전트인지 불명확
- 무엇을 보여줄지, 누구에게 보여줄지, 길이 전부 불명
- 스토리라인 구성 불명확

## 올바른 라우팅

1. 에이전트 이름과 핵심 기능 확인
2. 청중(내부/외부/투자자) 확인
3. 영상 길이 결정
4. 스킬 호출

## 수정 방향

"고객 문의 자동 분류 에이전트의 데모 영상을 투자자 발표용으로 만들어줄래? 시간: 60초, 청중: 투자자, 핵심 메시지: 30분 수동 분류 → 10초 자동 분류, 구성: Hook → 문제 정의 → 데모 → Before/After 비교 → CTA."
