# Bad Example — infographic-gif-creator 스킬

## 사용자 요청

"에이전트 아키텍처 그림 만들어줄래?"

## 거절 이유

- 정적 이미지인지 애니메이션인지 불명
- 어떤 아키텍처 패턴인지 불명
- 핵심 메시지가 불명확

## 올바른 라우팅

1. 시각화 형식 결정 (정적/애니메이션)
2. 대상 아키텍처 패턴 확인
3. 핵심 메시지 1문장 정의
4. 스킬 호출

## 수정 방향

"3-Tier 에이전트 아키텍처를 애니메이션 GIF로 설명해줄래? 사용자 입력 → Orchestrator → Sub-agent A/B 순차 등장으로 각 계층의 역할을 직관적으로 보여주고 싶어. README에 삽입 가능한 1280×720 GIF로."
