# Troubleshooting — cost-sim

## 1) 외부 API 가격 정보가 변동됨

**증상:**
- 공식 가격 페이지와 비용 시뮬레이션의 단가가 불일치
- 예: Brave Search가 $3에서 $3.50으로 인상됨

**확인:**
- 공식 가격 페이지 (Anthropic, OpenAI, API 제공사) 최신 정보 확인
- 2026-03 기준이 여전히 유효한지 검증

**조치:**
- "2026-03 기준 추정치"로 명시 + 공식 링크 제공
- 가격 변동이 발견되면 즉시 업데이트
- "구현 시 최신 가격 재확인 필수" 경고 추가

---

## 2) 스케일 시나리오에서 비선형 비용 증가

**증상:**
- 100명 사용자 시 비용이 1명의 200배 이상으로 증가
- 예: 1명 $10/월 → 100명 $2,500/월 (250배)
- 원인: Orchestrator 호출 폭증, 중복 요청 증가

**확인:**
- 비용 증가율이 사용자 증가율과 큰 차이 나는지 확인
- 오케스트레이션 레이어의 병렬 호출로 인한 비용 증가 여부

**조치:**
- "비선형 비용 증가 경고" 강조 표시
- 즉시 모델 라우팅 및 캐싱 전략 제안
- "100명 규모에서는 모델 라우팅으로 50% 이상 절감 가능" 구체적 수치 제시
- 오케스트레이션 최적화 (병렬 vs 순차, 중복 요청 제거) 권장

---

## 3) 단가 조회 불일치

**증상:**
- 웹 검색 결과와 공식 API 문서의 가격이 다름
- 또는 기준 기간이 불명확 (공시 날짜가 언제인가?)

**확인:**
- 공식 가격 페이지의 게시 날짜 확인
- 모델별/티어별 가격 차이 있는지 확인 (예: 엔터프라이즈 vs 스탠다드)

**조치:**
- 항상 공식 문서 링크 제공 (Anthropic, OpenAI)
- 게시 날짜 명시: "2026-03 기준"
- "가격은 공시일로부터 변동될 수 있으며, 최신 정보는 공식 API 문서 확인" 조건 추가
