# Good Example — cost-sim

## 사용자 요청
"고객 응대 에이전트를 GPT-4o로 만들려는데, 하루 1000건 처리 기준으로 월 비용이 얼마나 나올지 시뮬레이션해줘. 평균 대화 턴은 5회 정도야."

## 승인 이유
- 에이전트 타당성 평가 시 월간 비용을 추정해야 할 때
- 모델, 호출 빈도, 외부 API가 명확하게 제시됨
- 구현 전 비용 범위를 예측하여 예산 결정에 활용

## 예상 처리
1. 에이전트 프로파일링 (모델: GPT-4o, 호출: 1000건/일, 턴: 5회)
2. 토큰 비용 계산
   - 평균 입력 토큰 (per turn), 출력 토큰 (per turn) 추정
   - 월간 호출: 1000 × 30 = 30,000회
   - 월간 토큰 비용 산출
3. 스케일 시나리오 (1명 → 10명 → 100명)
4. 비용 최적화 전략 제안 (모델 라우팅, 캐싱 등)
5. 비용 KPI 및 build-or-buy 결정 연결
