## 예시 1: 핀테크 앱 (2030 직장인 대상)

### 타겟 프로파일
- 카테고리: 핀테크
- 대상: 2030 직장인
- 톤: 신뢰감 + 심플

### 선택된 레퍼런스

| 사이트 | 카테고리 | 적용할 패턴 | 선택 이유 |
|--------|----------|------------|----------|
| 토스 | 핀테크(KR) | 화이트 배경, 파란 CTA, 넓은 여백 | 국내 핀테크 표준, 신뢰감+친근함 균형 |
| 뱅크샐러드 | 핀테크(KR) | 데이터 시각화, 차트 컴포넌트 | 데이터 표현 레퍼런스 |
| Revolut | 핀테크(글로벌) | 다크 모드 옵션, 카드 UI | 글로벌 기준 제공 |
| Wise | 핀테크(글로벌) | 투명성 강조 UI, 수수료 표기 방식 | 투명성 패턴 |
| Linear | 생산성(글로벌) | 빠른 단축키, 개발자 미학 | 타겟 직장인의 다른 앱 참고 |

### 추출된 디자인 방향성

**레이아웃**: 카드 기반, 여백 충분, 정보 우선순위 명확. 1컬럼 모바일 우선.
  ❌ 정보를 한 화면에 모두 욱여넣는 밀도 높은 레이아웃
  ✅ 스크롤을 허용하되 한 섹션에 하나의 핵심 메시지

**컬러**: #FFFFFF 베이스, #3B82F6 브랜드 포인트(WCAG AA 4.6:1), #1F2937 텍스트.
  ❌ 여러 강한 색 동시 사용
  ✅ 파란색 하나를 CTA와 강조에만 사용 (80/15/5 비율 준수)

**타이포그래피**: 산세리프 2종(디스플레이/본문), 1.25 스케일. body 16px 기준.
**인터랙션**: 부드러운 transition(200ms ease), 호버 시 미세 음영.

### 디자인 방향성 결론
신뢰감(파란 포인트+넓은 여백)과 데이터 가독성(차트+카드)의 균형.
토스 수준의 심플함을 기준으로, 전문성을 데이터 시각화로 표현.

## 예시 2: 음식 배달 앱 (MZ세대 대상)

### 타겟 프로파일
- 카테고리: 푸드딜리버리
- 대상: MZ세대 (20-30대)
- 톤: 친근함 + 재미

### 선택된 레퍼런스

| 사이트 | 카테고리 | 적용할 패턴 | 선택 이유 |
|--------|----------|------------|----------|
| 배달의민족 | 푸드딜리버리(KR) | 연두+화이트, 일러스트, 유머 카피 | 국내 표준, MZ 친근감 |
| 당근마켓 | 소셜·커뮤니티(KR) | 오렌지, 동네 친근감, 신뢰 배지 | 신뢰+친근함 패턴 |
| Uber Eats | 푸드딜리버리(글로벌) | 음식 사진 대형, 빠른 주문 | 글로벌 음식 앱 UX |
| TikTok | 소셜·커뮤니티(글로벌) | 스크롤 몰입, 전체화면 | MZ 소비 패턴 |
| Instagram | 소셜·커뮤니티(글로벌) | 이미지 중심, 심플 | 음식 사진 레이아웃 |

### 추출된 디자인 방향성

**레이아웃**: 이미지 그리드, 빠른 스크롤, 카테고리 탭.
  ❌ 텍스트 중심의 리스트 UI
  ✅ 음식 사진이 2/3 이상 차지하는 카드 그리드

**컬러**: 연두(#A8E063) + 화이트 베이스, 따뜻한 웜톤 보조색.
  ❌ 차갑고 전문적인 블루 계열
  ✅ 식욕을 자극하는 웜톤+밝은 채도

**타이포그래피**: 둥근 산세리프, 굵은 헤딩, 캐주얼한 톤.
**인터랙션**: 탭 시 bounce 효과(300ms), 음식 카드 호버 시 1.03 확대.

### 디자인 방향성 결론
음식 사진 중심 이미지 레이아웃(TikTok/Instagram 참조)에 MZ 친근감(배달의민족 일러스트+둥근 폰트) 조합.
