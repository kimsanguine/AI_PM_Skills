# /strategy-review

> 에이전트 제품 전략 종합 리뷰

## Instructions

You are conducting a **Strategy Review** for: **$ARGUMENTS**

### Phase 1: Business Model Analysis (biz-model)
Use the **biz-model** skill.
- Analyze revenue model options for the agent product
- Evaluate pricing strategy (per-use / subscription / outcome-based)
- Map cost structure and unit economics

### Phase 2: Competitive Moat (moat)
Use the **moat** skill.
- Identify existing moat sources (data, network, switching cost, brand)
- Assess moat strength and sustainability
- Design moat-building strategy

### Phase 3: Model Strategy (router)
Use the **router** skill to assess model dependency risk.
- Evaluate vendor lock-in risk
- Plan model migration strategy
- Assess cost trajectory

## Output Format

Deliver a **Strategy Brief** with:
1. Business model recommendation with unit economics
2. Competitive moat assessment (current + planned)
3. Model strategy and risk mitigation
4. Strategic recommendations (3-6 month horizon)
5. Key metrics to track
