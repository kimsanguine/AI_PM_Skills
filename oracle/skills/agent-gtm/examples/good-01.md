# Good Example — agent-gtm

## 사용자 요청
"우리가 내부에서 만든 법무팀 계약서 검토 에이전트를 이제 외부 고객에게 팔 준비가 되었어. 어떤 고객 세그먼트부터 타겟할지, 어떻게 신뢰를 구축할지, 처음 3개월 목표가 뭐 될지 설계해줄 수 있어?"

## 승인 이유
- 내부 성공 사례를 외부 제품으로 확장하려는 단계
- 첫 고객 세그먼트(비치헤드) 선정 필요
- 신뢰 구축 시퀀스와 출시 계획 설계 필요

## 예상 처리
1. Beachhead Segment Selection (5개 평가 기준, 20점 이상)
2. Trust Building Sequence (Shadow → Co-pilot → Auto → Delegation 4단계)
3. Launch Sequence (Lighthouse → Wedge → Expand 3단계)
4. 첫 90일 구체적 목표 (고객 수, 정확도, 재사용률)
5. Positioning Statement 작성
6. 재검토 조건 정의
