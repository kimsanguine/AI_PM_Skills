# Bad Example — opp-tree

## 사용자 요청
"우리 팀 OKR 작성 좀 도와줘. 이번 분기 목표는 MAU 20% 성장이야."

## 거절 이유
- 에이전트 기회 탐색이 아닌 팀 전체의 비즈니스 OKR 설정
- opp-tree는 "에이전트로 자동화할 수 있는 기회"를 찾는 스킬
- 비즈니스 전략/OKR은 다른 스킬 범위

## 올바른 라우팅
- OKR/전략 기획: 별도 스킬 (muse/strategy 플러그인)
- 에이전트 기회 발굴: opp-tree 스킬

## 수정 방향
"우리가 MAU 20% 성장을 달성하려면, 고객 온보딩을 빨리 해야 하는데 수동 부분이 많아. 어디를 에이전트로 자동화하면 빠를까?" → opp-tree로 라우팅 가능
