RECOMMENDATIONS
================================================================================

USE WITH_SKILL when:
  ✓ Strict architectural structure required
  ✓ Complex agents with multiple failure modes
  ✓ Team/portfolio standardization needed
  ✓ Quality > cost/speed (production PRDs)
  ✓ Agent specifications for implementation
  ✓ Documentation must include explicit anti-goals

USE WITHOUT_SKILL when:
  ✓ Rapid MVP/prototype PRDs
  ✓ Cost-sensitive projects
  ✓ Narrative/flowing prose preferred
  ✓ Using cached results from prior requests
  ✓ Non-standard agent architectures
  ✓ Want more comprehensive user-facing docs
