# Good Example — agent-demo-video 스킬

## 사용자 요청

"고객 문의 자동 분류 에이전트의 데모 영상을 투자자 발표용으로 만들어줄래? Slack에서 문의가 들어오면, 자동으로 분류하고 담당자에게 배정하는 과정을 보여줘야 해. 60초 분량으로."

## 승인 이유

- 에이전트의 동작과 가치를 영상으로 설득력 있게 전달 필요
- Remotion 기반으로 재현 가능하고 수정 용이한 영상 필요
- 화면 녹화 + 아키텍처 애니메이션 + 성과 비교 포함

## 예상 처리

1. 스토리라인: Hook(5s) → 문제(10s) → 데모(30s) → 성과(10s) → CTA(5s)
2. Composition: 각 씬별 duration(프레임 단위)과 트랜지션
3. 타임라인: Sequence 구조로 스크린 레코딩 + 텍스트 애니메이션
4. 에셋 통합: 배경음 + 자막 + Before/After 비교 차트
5. QA: 1920×1080, 30fps 렌더, 오디오-비주얼 싱크 확인

## 최종 결과물

60초 데모 영상으로 투자자에게 에이전트의 가치를 명확히 전달
