# Bad Example — pptx-ai-slide 스킬

## 사용자 요청

"발표 자료 만들어줄래?"

## 거절 이유

- 발표 주제 불명확
- 청중/목적 미정의
- 분량 미지정

## 올바른 라우팅

1. 발표 주제(에이전트명) 확인
2. 청중(C-Level/PM/엔지니어) 확인
3. 핵심 메시지 3개 정의
4. 스킬 호출

## 수정 방향

"고객 문의 자동 분류 에이전트를 C-Level에 피칭할 발표 자료를 만들어줄래? 핵심 메시지: 1) 비용 절감(월 $N), 2) 응답 속도 개선(30분→10초), 3) 고객 만족도 상향. 7장 이내로 구성해줘."
