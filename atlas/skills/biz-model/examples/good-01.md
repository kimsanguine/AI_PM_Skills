# 좋은 예제 — Business Model

## 사용자 요청
"우리 고객 서비스 자동화 챗봇이 상담사 40% 시간을 절감한다는 걸 알았는데, 이걸 바탕으로 어떤 가격 모델과 수익 구조로 가야 할까?"

## 승인 이유
- 고객이 창출하는 **구체적 가치**(시간 절감, 비용 감소)를 알고 있음
- 에이전트의 **변수 비용 구조**(LLM API, 인프라)를 파악 가능
- **가격 책정 전략 수립** 필요

## 예상 처리
1. 가치 창출 분석: 월 64시간 절감 × 15,000원 = 960,000원
2. 비용 구조: LLM $0.05/건, 월 2,000건 = $100, 인프라 $500 등
3. 가격 모델: 월정액 + 초과분 하이브리드 모델 선택
4. 이윤율: Gross Margin >70% 검증
5. 고객 검증: 실제 또는 예상 고객과 가격 타당성 확인

## 처리 결과 예시
```
Business Model Canvas:

Value: 월 960,000원 절감 (상담사 40% 시간)

Cost Structure:
- CPE (Cost Per Execution): $0.05/건
- 월 2,000건: $100 LLM
- 인프라: $200
- 지원: $50
- 총 비용: $350

Pricing: $800/월 (기본 1,500건 포함 + 초과 $0.20)
Revenue: $800/월
Gross Margin: ($800 - $350) ÷ $800 = 56%

Payback: 고객이 월 960,000원 절감 중 800,000원만 우리에게 지불
→ ROI 120배 (고객 입장 매력적)
```
